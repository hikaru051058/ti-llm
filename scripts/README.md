# Scripts

Place helper utilities here, e.g.:
- `deploy-gateway.sh` to build/deploy the CDK stack
- `set-factory-secret.sh` to create/update the factory secret
- `push-ti-chat.sh` to generate/push the TI CHAT program (requires tilp/ticonv)
- `provision-test.sh` to call `/provision`
- `ask-test.sh` to call `/ask`
- log tail helpers
